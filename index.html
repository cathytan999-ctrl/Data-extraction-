Why people will use this

Most property pages only show listings. This suite explains timing, relative value, and risk in one place. That makes it useful for owner-occupiers, investors, agents, and anyone comparing projects in the same district.

1
Understand the market
See whether the broader market is accelerating, slowing, cooling, or stabilising using quarterly market momentum.
2
Benchmark the district
Use district resale trend and median PSF to anchor realistic pricing, not just marketing stories.
3
Judge the project
See whether the project trades above or below district and whether it has outperformed or underperformed over time.
4
Make the call
Turn pricing, performance, and market momentum into a deal score and smart signal that is easy to explain.

Data sources

URA / data.gov.sg

Macro market momentum comes from the official quarterly private residential property price index published on Singapore's open data portal. It includes long history and supports Overall, CCR, RCR, and OCR series.

URA transaction search

District and project benchmarking comes from exported private residential transaction data. The transaction-search export covers the last 60 months and supports resale, sub-sale, and new sale filtering.

Processing layer

Exported files are transformed into JSON so the dashboard can run on a static site such as GitHub Pages without a backend.
